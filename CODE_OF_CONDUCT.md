# Código de Conducta

Este proyecto adopta el [Código de Conducta del Contribuyente](https://www.contributor-covenant.org/es/version/2/0/code_of_conduct/).

Resumen:
- Sé respetuoso y profesional.
- Los comportamientos ofensivos, discriminatorios o de acoso no serán tolerados.
- Si eres testigo o víctima de un incumplimiento, contacta a los mantenedores vía issue privado o al correo del autor del proyecto.

Detalles: https://www.contributor-covenant.org/es/version/2/0/code_of_conduct/
